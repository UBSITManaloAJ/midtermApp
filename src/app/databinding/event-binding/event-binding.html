Event Binding

Listen and respond to user-triggered DOM events using (event)="handler" syntax. One-way binding from the view to the component.

📌 How it works

Parentheses tell Angular: listen for this DOM event and execute the expression when it fires. The special $event variable holds the native Event object.

// Click event
<button (click)="onButtonClick()">Click Me</button>

// Keyboard event — pass $event for key details
<input (keyup)="onKeyPress($event)">

// Form submit — prevent page reload
<form (submit)="onFormSubmit($event)">
