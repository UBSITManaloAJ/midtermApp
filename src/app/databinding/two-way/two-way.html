Two-Way Binding

Synchronize data between component and template simultaneously using [(ngModel)] — the "Banana in a Box" syntax. Requires FormsModule in imports.

📌 How it works

[(ngModel)] is syntactic sugar that combines both directions:

// [(ngModel)] is shorthand for:
[ngModel]="prop"               ← Property Binding: component → view
(ngModelChange)="prop=$event"  ← Event Binding: view → component

// Combined:
<input [(ngModel)]="twoWayName">
