Property Binding

Set DOM element properties dynamically using [property]="expression" syntax. One-way binding from the component to the view.

📌 How it works

Square brackets tell Angular: evaluate the right-hand side as a TypeScript expression and assign its result to the DOM property named inside the brackets.

// Sets the JS property, NOT the HTML attribute
<img [src]="imageSource" [alt]="imageAlt">
<button [disabled]="isButtonDisabled">Click Me</button>
<input [placeholder]="inputPlaceholder">
❌ Without [ ] disabled="isButtonDisabled" Always disabled — treats value as a string
✅ With [ ] [disabled]="isButtonDisabled" Evaluates to true/false — dynamic
